Ocupação Hospitalar

Descrição

O conjunto de dados de ocupação hospitalar é um dado de saúde obtido do Ministério da Saúde do Brasil por meio do Departamento de Atenção Especializada à Saúde (SAES), (Ministério da Saúde, 2020b). Este banco de dados começou em abril de 2020 devido à pandemia COVID-19. A partir de então foi implantado o cadastro de internações e ocupação dos leitos por meio do Módulo de Admissões do Sistema ESUS Notificação-SUS. O banco de dados contempla a quantidade de leitos disponíveis para cada unidade de saúde, quantos foram ocupados por pacientes com casos suspeitos ou confirmados de COVID-19 (Ocupação SRAG/COVID-19) e quantos foram ocupados por pacientes de outra especialização médica.

Informação de acesso a dados

Os dados estão licenciados sob a Creative Commons Attribution License cc-by (versão 4.0), ver (Ministério da Saúde, 2020b). Além disso, o conjunto de dados está disponível ao público e publicado pelo Ministério da Saúde do Brasil. Portanto, não é necessária a aprovação de um comitê de ética para o uso desses dados, de acordo com as Resoluções 466/2012 e 510/2016 (artigo 1º, incisos III e V) do Conselho Nacional de Saúde (CNS), Brasil.

Métodos de coleta de dados

Um código python está disponível em nosso diretório GitHub para baixar os dados do OpenDatasus, veja detalhes em Github. A base de dados é atualizada semanalmente no nosso projeto, sendo uma versão associada à sua data de atualização. Informações adicionais sobre os métodos de coleta de dados podem ser encontradas em (Ministério da Saúde, 2020b) e Github.

Informações específicas de dados para ocupação hospitalar

O conjunto de dados de ocupação hospitalar tem um total de 21 colunas e mostrou um total de 1.164.623 registros (linhas) e um tamanho de 219Mb na última atualização de 11 de outubro de 2021. Um código com mais detalhes sobre as variáveis, processamento de dados e métodos de análise é apresentado em nosso diretório Github.
